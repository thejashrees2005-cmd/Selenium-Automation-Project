# Selenium-Automation-Project
Hands-on Selenium WebDriver automation practice covering UI testing across multiple real-world style websites.

## Tech Stack
- *Language:* Java
- *Automation Tool:* Selenium WebDriver
- *Testing Framework:* TestNG
- *Build Tool:* Maven
- *Reporting:* Extent Reports, TestNG Reports

## Projects Covered

| Project | Site | Test Cases | Key Concepts |
|---|---|---|---|
| LetCode | letcode.in | 9 | Forms, dropdowns, alerts, button actions |
| DemoQA | demoqa.com | 7 | Text boxes, form fields, link extraction |
| Herokuapp | the-internet.herokuapp.com | 8 | A/B testing, basic auth, drag-and-drop, screenshots |
| Blogspot | testautomationpractice.blogspot.com | 11 | Forms, calendar, double-click, drag-and-drop, scrolling |
| PracticeTestAutomation | practicetestautomation.com | 10 | Login flow, form submission, navigation |
| Toscrape | books.toscrape.com | 14 | Pagination, category navigation, cart actions |
| RahulShettyAcademy | rahulshettyacademy.com | 9 | Radio buttons, dropdowns, checkboxes, suggestions |
| UITestingPlayground | uitestingplayground.com | 6 | Scrollbars, login, mouse-over actions |

*Total Test Cases:* 74

## Skills Demonstrated
- Locating elements using XPath
- Handling alerts, dropdowns, checkboxes, and radio buttons
- Mouse actions (drag-and-drop, double-click, right-click, hover)
- Screenshot capture during execution
- Page navigation and scrolling automation
- Test execution and reporting with TestNG and Maven
